## Getting Started

First, install Tauri:

(https://tauri.app/v1/guides/getting-started/prerequisites)

Run the development server:

```bash
npm run tauri dev
# or
yarn tauri dev
# or
pnpm tauri dev
# or
bun tauri dev
```
